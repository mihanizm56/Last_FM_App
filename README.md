
Компоненты:
FilterGenres - поле фильтрации по жанрам - вид:тэг select
LinkItem - ссылка - вид:тэг Link
Logo - ссылка - вид:тэг Link
MiddleImage - изображение средних размеров - вид:тэг div со style={{backgroundColor}}
PaginationButton - кнопка - вид:тэг div, внутри лежит Link
Paragraph - параграф - вид:тэг р
PlayButton - кнопка - вид:тэг div, внутри лежит img
SearchField - поле ввода фильтра - вид:тэг input
SmallImage - изображение малых размеров - вид:тэг div со style={{backgroundColor}}
TitleH2 - Заголовок - вид:тэг h2

Модули:
AlbumBigBlock - блок с Заголовком названия альбома
ArtistBox - блок для фотки на заднем плане и ссылки внутри
SongsBigBlock - основной блок для "страницы" Треки
TrackForAlbum - блок трека для "страницы" Альбом
TrackForChart - блок трека для блока "Last FM Chart"
TrackForList - блок трека для "страницы" Треки
